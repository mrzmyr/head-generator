head-generator
==============
Generates the complete `<head>` of a HTML5 page for you.

[Check it out!](http://mrzmyr.github.io/head-generator)

### Support

* doctype
* language 
* content-type 
* favicon
* ie options
* viewport 
* iOS options
* open graph 
* item props
